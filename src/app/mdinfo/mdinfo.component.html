Cómo usar Markdown

Una guía rápida para principiantes

Encabezados

Para crear un encabezado, simplemente añade uno o más signos de número (#) al comienzo de una línea. El número de signos de número indica el nivel del encabezado. 

# Encabezado de nivel 1
  ## Encabezado de nivel 2
  ### Encabezado de nivel 3

Énfasis

Puedes añadir énfasis a tu texto usando asteriscos o guiones bajos. Usa un solo asterisco o guión bajo para cursiva y dos para negrita. 

*cursiva* o _cursiva_
  **negrita** o __negrita__

Listas

Para crear una lista, simplemente añade guiones, asteriscos o números al comienzo de las líneas. 

- Elemento de lista
  * Elemento de lista
  1. Elemento numerado

Enlaces

Puedes crear enlaces envolviendo el texto del enlace en corchetes y la URL en paréntesis. 

[Texto del enlace](https://www.ejemplo.com)

Imágenes

Para añadir una imagen, usa una exclamación seguida de corchetes con el texto alternativo y paréntesis con la URL de la imagen. 

![Texto alternativo](https://www.ejemplo.com/imagen.jpg)
